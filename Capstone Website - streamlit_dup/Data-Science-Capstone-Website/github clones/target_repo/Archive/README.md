# Python Snake Game

This is a snake game in python. The default characters are as below.

- Snake: `#`
- Food: `*`
- Background: `.`

## How to play

WASD or arrow keys for snake movement. Q to quit. Any other key to pause. Press one of the movement keys to resume.

## Running the game

```bash
python3 snake.py [flags]
```

### Flags

See `snake.py --help`
